# 🎯 Engenharia de Prompt: Otimização e Refinamento Iterativo

## 📝 Descrição do Projeto
Este diretório contém a documentação de uma atividade prática avançada em Engenharia de Prompt. O foco do laboratório foi transpor a barreira de usuário comum para atuar no desenvolvimento de instruções de alta precisão para Large Language Models (LLMs). 

A premissa do desafio exigiu a geração de um *output* específico (texto ou imagem) utilizando o modelo **Qwen AI**, sob uma restrição técnica rigorosa: atingir a exatidão do resultado estipulado ("Alvo") em um limite máximo de 5 iterações.

## 📂 Conteúdo do Diretório
* `Prompt 1 enviado.pdf`: Documento contendo o registro da evolução iterativa, demonstrando o histórico de prompts enviados, as respostas obtidas e a análise crítica aplicada entre cada tentativa até a conclusão do objetivo.

## 🚀 Metodologia de Execução
O projeto foi executado através de um framework de iteração contínua, estruturado em duas fases analíticas para cada prompt submetido:
1. **Avaliação Crítica:** Análise minuciosa do *output* gerado pela IA. Identificação de falhas de contexto, desvios de tom, alucinações ou diretrizes ignoradas em relação ao "Alvo".
2. **Integração e Refinamento:** Utilização do erro da iteração anterior como base de dados analítica para otimizar o prompt seguinte. Aplicação de ajustes semânticos, estruturais e adição de restrições para guiar o modelo à perfeição.

## 📊 Resultados e Aprendizados
O laboratório evidenciou que a eficácia de um modelo de IA é diretamente proporcional à qualidade e estruturação da instrução fornecida. Os principais aprendizados técnicos incluem:
* **Eficiência de Instrução:** A importância de fornecer contextos exatos e diretrizes restritivas para reduzir o nível de abstração e entropia na resposta da máquina.
* **Diagnóstico de Erro (Troubleshooting):** A capacidade de diagnosticar *por que* o modelo falhou e ajustar parâmetros específicos no prompt seguinte, evitando a repetição do erro no ciclo de refinamento.
* **Otimização de Ciclos:** O desenvolvimento da habilidade de convergir para a solução ideal rapidamente, gerenciando a complexidade da solicitação dentro do limite estrito de tentativas impostas pelo cenário.
